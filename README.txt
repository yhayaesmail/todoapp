# Todo App

**Date:** 17/09/2025

## Overview

This is a **Todo App** and it is my **first project** in web development using **Node.js and Express.js**. The app helps users manage their daily tasks in a simple and organized way. In this project, I also used **Prisma** for database management with **PostgreSQL** and **Docker** for containerization to better understand deployment and scalability.

## Features

* Add new tasks
* View existing tasks
* Mark tasks as completed
* Delete tasks

## Technologies Used

* Node.js
* Express.js
* Prisma (ORM for database management)
* PostgreSQL (database)
* Docker (for containerization)
* HTML, CSS, JavaScript

## Getting Started

1. Clone the repository:

   ```bash
   git clone <your-repo-url>
   ```
2. Install dependencies:

   ```bash
   npm install
   ```
3. Run the app locally:

   ```bash
   npm start
   ```
4. Open your browser and go to:

   ```
   http://localhost:3555
   ```

## Notes

This is my first project, so the focus was on learning the basics of backend development and connecting it to a simple frontend interface. Using Prisma with PostgreSQL allowed me to manage the database efficiently, and Docker helped me understand how to containerize and run the application consistently across different environments.
